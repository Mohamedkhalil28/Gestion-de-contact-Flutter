# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-02

### Added
- Initial release of Gestion de Contact Flutter
- MongoDB integration for contact storage
- Material Design 3 UI implementation
- Cross-platform support (Android, iOS, Web, Desktop)
- CRUD operations for contacts (Create, Read, Update, Delete)
- Simple authentication system
- Responsive design with modern UI components

### Features
- Add new contacts with name, phone, and email
- Edit existing contact information
- Delete contacts from database
- View all contacts in a clean list interface
- Secure login with password authentication

### Technical Details
- Built with Flutter 3.x and Dart
- Uses mongo_dart package for MongoDB connectivity
- Implements proper error handling and async operations
- Follows Flutter best practices and clean architecture

### Dependencies
- flutter: SDK
- mongo_dart: ^0.10.5
